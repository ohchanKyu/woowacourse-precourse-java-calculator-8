# java-calculator-precourse

## 개요
쉼표(,) 또는 콜론(:)을 구분자로 사용하거나, 사용자 정의 구분자를 지정하여 입력된 문자열에서 숫자를 추출하고 그 합을 계산하는 기능을 구현합니다.
